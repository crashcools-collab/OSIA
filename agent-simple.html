Agent Simple

// CE QUE C'EST

Un agent IA simple = un script qui orchestre plusieurs prompts dans un ordre logique pour accomplir une tâche précise et répétitive. Tu lances, il fait, t'oublies.

// EXEMPLES CONCRETS

  • 🔍 Agent veille : scrape 10 sites + résume en 1 brief Markdown quotidien
  • 📝 Agent rédacteur : génère 1 article SEO complet (brief + outline + rédac + meta)
  • 📧 Agent répondeur : trie tes mails et propose des réponses prêtes
  • 📊 Agent analyste : lit un CSV et produit un rapport texte
  • 🎨 Agent créatif : génère 10 idées de posts à partir d'1 thème

// CE QUE TU RECEVES

  • ✓ Script Python prêt à exécuter
  • ✓ Documentation : input, output, configuration
  • ✓ Fichier .env exemple pour les API keys
  • ✓ 1 session de configuration ensemble (Zoom)

// LIMITES

Un agent simple ne décide pas seul de la suite. Il suit un script linéaire. Pour un agent autonome qui prend des décisions et s'adapte, regarde l'Agent Supérieur →
