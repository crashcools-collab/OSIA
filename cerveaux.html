// CE QUE C'EST

Un Cerveau OSIA = un orchestrateur central qui coordonne plusieurs agents IA spécialisés. Tu donnes une mission complexe au cerveau, il la décompose, il assigne chaque sous-tâche au bon agent, il synthétise le résultat final.

C'est ton chef d'orchestre numérique. Là où un agent fait UNE tâche, un cerveau pilote 5, 10, 30 agents en parallèle.

// EXEMPLES CONCRETS

  • 🎯 Cerveau commercial : pilote agents qualif + cold mail + CRM + relance → tu gères 200 leads/mois sans effort
  • 📞 Cerveau support : route les demandes clients vers agent SAV, agent technique, agent commercial selon le sujet
  • 📊 Cerveau analyste : reçoit données → distribue à agent SQL, agent visualisation, agent rapport → livre un dashboard complet
  • ✍️ Cerveau contenu : un brief → orchestre agent recherche + agent rédaction + agent SEO + agent image → article complet livré
  • 🛒 Cerveau e-commerce : pilote agent stock + agent prix + agent description + agent SAV pour ta boutique
  • 👨‍💼 Cerveau RH : agent tri CV + agent entretien + agent contrat + agent onboarding

// CE QUE TU REÇOIS

  • ✓ Codebase Python du cerveau (architecture modulaire)
  • ✓ 2 agents inclus au choix, intégrés au cerveau
  • ✓ Dashboard de monitoring (qui fait quoi, quand, combien ça coûte)
  • ✓ Interface web pour parler au cerveau
  • ✓ Logs détaillés de chaque décision et appel d'agent
  • ✓ Doc technique + schéma d'architecture
  • ✓ 3 sessions de paramétrage (Zoom)
  • ✓ 30 jours de support post-livraison
  • ✓ Possibilité d'ajouter d'autres agents après

// POUR QUI

Tu as un processus métier complexe qui implique plusieurs étapes (qualifier, traiter, livrer, suivre) et tu veux automatiser TOUT l'enchaînement, pas juste une étape.

Un agent simple ≠ un cerveau. L'agent fait UNE chose. Le cerveau coordonne plusieurs choses. Si ton besoin tient en un seul prompt, prends un agent. Si tu veux qu'OSIA orchestre, prends un cerveau.
